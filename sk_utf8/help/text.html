Pomôcka na písanie textu

Písanie textov o Moodle v mnohom spĺňa vaše očakávania. Navyše máte ešte možnosť do textov použiť "smajlíkov", URL adresy a niektoré HTML tagy.

URL adresy

Každý text začínajúci s www. alebo http:// sa automaticky zmení na klikací odkaz.

Napríklad: www.yahoo.com alebo http://curtin.edu

HTML tagy

Na zvýraznenie niektorých častí vášho textu môžete použiť limitovanú sadu HTML tagov.
HTML tagyČinnosti
<b> bold </b> tučný text
<i> italic </i> text kurzívou
<u> underline </u> podčiarknutý text
<font color="green"> vzorka </font> vzorka
<ul> <li>jeden</li> <li>dva</li> </ul>
